# ABC 量化交易策略文档 (V1)

## 一、策略概述

本策略基于"股票是筹码，背后是人"的核心认知，利用其他散户的情绪波动（贪婪与恐慌），通过底仓+浮仓的两层结构，在风险可控的前提下，逐步回收本金，最终实现零成本持仓。

**策略定位**：低频、日内波段、严格风控  
**持仓周期**：浮仓隔夜，底仓长期  
**操作频率**：一个月一只票，空仓是常态

---

## 二、选股规则（静态筛选）

标的池每周更新一次，每次只操作一只票。

| 条件 | 标准 | 数据来源 |
|------|------|----------|
| 股东人数 | > 10万户 | 季报 |
| 人均持股 | < 5万元 | 流通市值 / 股东人数 |
| 日均换手率 | 5% – 15%（过去20日） | 日线数据 |
| 日均成交额 | > 2亿元 | 日线数据 |
| 波动区间 | 月内最高/最低 > 1.2 | 日线数据 |
| 排除项 | 庄股、量化高频股（换手率>20%且走势机械）、新股 | 人工/规则 |

**选股补充**：底仓建立前，需观察盘口活跃度3天以上，确认非死水。

## 三、仓位结构

| 类型 | 比例 | 用途 | 操作频率 |
|------|------|------|----------|
| 底仓 | 10% | 本金锚点，长期持有 | 建立后不动，除非触发止损或30天无交易 |
| 浮仓 | 每日最多5手（约1万元） | 顺势波段，赚取利润 | 每日可交易，隔夜了结 |
| 总仓位上限 | ≤ 45% | 风险控制 | 盘中动态监控 |

## 四、底仓规则

### 4.1 建立条件（两个必须同时满足）

1. **散户贪婪程度高**（当天14:30统计）：
   - 买盘堆积次数（买一至买三挂单量 > 过去5分钟均值的2.5倍）超过当日阈值
   - 追高单子量（市价买入总量）超过当日阈值
   - （具体阈值后续回测确定）

2. **筹码峰集中**：
   - 分析当天与过去30天筹码峰变化，显示数量最多的价格区间集中度提高
   - 结合股东人数确认散户占比较高

### 4.2 止盈止损

| 条件 | 执行 |
|------|------|
| 盈利 +10%（从建仓价算） | 清掉全部仓位（底仓+所有浮仓） |
| 亏损 -10%（从建仓价算） | 清掉全部仓位（底仓+所有浮仓） |

### 4.3 持有期限

- 底仓建立后，30天内无任何浮仓交易机会 → 清掉底仓，重新评估该票

## 五、浮仓规则

### 5.1 买入条件

**买入时间**：仅限 **14:30 之后**

**启动条件**：委比进入 **40% – 60%** 区间

**执行方式**：
- 一手一手买入（100股/手）
- 每手间隔至少15分钟
- 同一价位附近（±0.5%）最多买3手
- 当日累计买入不超过5手
- 顺势而为，不追高（价格位置在中低位优先）

**买入后的确认**：买入后若股价跌破该手成本-3%，暂停对该票的买入，直到价格稳定或止损。

### 5.2 卖出条件

**卖出时间**：9:30 之后，随时

**启动条件**：委比进入 **-40% – 60%** 区间

**止盈硬约束**：
- 每手净收益 ≥ 5%（扣手续费后，对应股价涨幅约5.3%）
- 达到即卖，不等

**止损硬约束**：
- 每手亏损 ≥ -6.5%（从该手买入价算）
- 达到即卖，不等

**执行方式**：
- 一手一手卖出
- 有信号即出，不等待
- 若全天无信号，收盘前卖出当日可卖仓位的50%

### 5.3 集合竞价辅助决策

| 竞价情况 | 当日卖出策略 |
|----------|--------------|
| 高开（>1.5%）+ 放量（匹配量 > 昨日成交额5%） | 优先等冲高 |
| 高开 + 缩量（匹配量 ≤ 昨日成交额2%） | 开盘先出可卖仓位的1/3 |
| 低开（<-1.5%）+ 放量 | 观察3分钟，不能拉回则止损 |
| 低开 + 缩量 | 正常按盘口信号出 |
| 极端情况（涨跌停开、巨量异常） | 暂停新开仓，只处理持仓 |

## 六、委比口诀辅助信号（盘口情绪识别）

| 口诀 | 信号含义 | 模型操作 |
|------|----------|----------|
| 委比正又高，放量价上冲 | 真强势，贪婪盘进场 | 浮仓卖出 |
| 委比高为正，缩量价不动 | 假买盘，主力诱多 | 不买，考虑卖 |
| 委比负且低，放量价下砸 | 真抛压，恐慌盘出逃 | 浮仓买入（接恐慌） |
| 委比低为负，缩量价企稳 | 抛压尽，反弹可期待 | 观察买点 |
| 委比忽大变，撤单又重挂 | 骗线多，假信号 | 不动 |
| 价涨委比降，量缩要提防 | 涨不动，随时回调 | 浮仓卖出 |
| 价跌委比升，量增有承接 | 跌放缓，分批可低吸 | 浮仓买入 |

**融合方式**：上述口诀作为盘口情绪的辅助识别工具，不单独作为操作依据，需结合委比区间和止盈止损条件综合判断。

## 七、风控规则

### 7.1 触发即执行（无条件）

| 触发条件 | 执行动作 |
|----------|----------|
| 底仓亏损 -10% | 清掉全部仓位（底仓+浮仓） |
| 浮仓单笔亏损 -6.5% | 止损该手 |
| 账户周亏损 -6.5%（以周一开盘为基准） | 清仓，停止交易至少3个交易日 |
| 浮仓周胜率 < 60%（交易笔数 ≥ 3） | 清仓，停止交易，复盘 |

**胜率定义**：盈利笔数 / 总浮仓交易笔数（底仓不参与）

### 7.2 强制休息

- 连续两次触发账户周止损或胜率周止损
- 或底仓止损后
- 休息至少3个交易日，复盘问题后再启动

## 八、收益核算与本金回收

### 8.1 核算方式

- **浮仓**：每笔独立核算已实现盈亏
- **底仓**：浮动盈亏单独记录
- **账户总资产**：底仓市值 + 浮仓现金 + 已实现盈亏

### 8.2 本金回收

- **回收前**：账户总资产 < 初始本金 → 所有风控规则生效，底仓与浮仓一视同仁
- **回收成功**：账户总资产 ≥ 初始本金 → 底仓变为零成本持仓
- **回收后**：底仓可长期持有；若账户总资产再次跌破本金，重新进入回收前状态

### 8.3 最终复盘

每只票完全清仓后（底仓+所有浮仓)，核算该票总盈亏，用于判断是否值得再次进入，以及操作纪律执行情况。

| 股东人数 | 季报数据 |
| 交易日历 | T+1 限制 |

## 十一、回测要求

| 项目 | 要求 |
|------|------|
| 回测周期 | 至少3个月，建议1年 |
| 初始资金 | 10万元 |
| 手续费 | 买入万1.5，卖出万1.5+印花税千1（合计约0.13%每回合） |
| 滑点 | 0.01元 或 0.02% |
| 交易时间 | 连续竞价时段（9:30-11:30, 13:00-15:00） |
| T+1限制 | 当天买入不能卖出 |
| 输出指标 | 总收益率、年化收益率、胜率、盈亏比、最大回撤、夏普比率、交易次数 |
